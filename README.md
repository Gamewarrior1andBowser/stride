# Stride – Footwear E-Commerce Website

Stride is a modern and visually appealing footwear e-commerce website created as part of our final project. The website showcases a wide range of shoes from top global brands as well as our own in-house **Stride** footwear collection. The goal of this project is to provide users with a clean, smooth, and intuitive shopping experience while demonstrating our skills in HTML, CSS, JavaScript, design, and responsive layout development.

## Link to Shop:- [Click here](https://gamewarrior1andbowser.github.io/stride/)


## **Overview**
The website provides customers with an easy and stylish way to browse shoes for men, women, and kids. It features a hero banner, product highlights, brand displays, category-based browsing, and an about section that introduces the purpose and values of Stride. Stride combines major global brands like Nike and Adidas with our own in-house designs, giving users a wide range of styles to choose from.


## **Homepage**
The homepage welcomes visitors with a clean and friendly design. It gives a clear idea of what Stride is about and helps users quickly get where they want to go. It makes it easy to start shopping or learn more about the brand, all while keeping the page simple and visually appealing
